Some security advantages:

Minimized attack surface

  • musl libc or muslc: Cleaner codebase, less bloated than glibc, decent exploit mitigations, more secure mallocng.
  • busybox: lightweight, less attack surface, the code is still bad (at least busybox is better than coreutils)
  • Based on minimalism, packages are splitted, you only install anything you need
